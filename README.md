# 🎉 शुभ विवाह आमंत्रण (Sadi Invitation Card)

## 📱 About / विवरण

यह एक **Interactive Digital Wedding Invitation Card** है जो HTML, CSS और JavaScript से बनाया गया है। यह आधुनिक, आकर्षक और उपयोगकर्ता के अनुकूल है।

**A beautiful and interactive digital wedding invitation card built with HTML, CSS & JavaScript with premium features and animations.**

---

## ✨ Features / विशेषताएं

### 🎨 Design Features
- ✅ **Beautiful Cover Page** - सुंदर envelope डिज़ाइन वाला कवर पेज
- ✅ **Responsive Design** - सभी डिवाइस पर परफेक्ट दिखता है (Mobile, Tablet, Desktop)
- ✅ **Gradient Effects** - प्रीमियम gradient और shadow effects
- ✅ **Glowing Elements** - सभी elements पर golden glow
- ✅ **Professional Layout** - विवाह आमंत्रण के लिए traditional और modern का mix

### 🎭 Interactive Features
- 🎆 **Fireworks Animation** - बटन क्लिक पर रंगीन fireworks
- 🌸 **Floating Flowers** - कवर पेज पर गिरते हुए फूल
- 🎵 **Background Music** - संगीत टॉगल बटन (Bottom-right)
- ✅ **RSVP Button** - अतिथि RSVP दर्ज कर सकते हैं
- 💬 **WhatsApp Share** - एक क्लिक से WhatsApp शेयर करें

### ⏱️ Time & Events
- ⏳ **Countdown Timer** - शादी तक लाइव गिनती
  - दिन / घंटे / मिनट / सेकंड
- 📅 **Event Schedule** - सभी समा���ोह की तारीख
  - मण्डप / हल्दी
  - मायन / संगीत
  - बारात आगमन

### 📍 Additional Features
- 📍 **Map Location** - Google Map लिंक के साथ location
- 📸 **Photo Gallery** - शादी की तस्वीरों का gallery
- 💒 **Couple Information** - दूल्हा-दुल्हन की जानकारी
- 👨‍👩‍👧‍👦 **Family Members** - परिवार के सदस्यों की list
- 📝 **Shayari** - विवाह से संबंधित शायरी

### 🎬 Animation Effects
- 🎊 **Welcome Page** - 5 सेकंड का welcome animation
- ✨ **Fade & Scale** - Smooth fade-in और scale animations
- 🎪 **Bounce Effects** - Elements पर bounce animation
- 🌺 **Float Animation** - Flowers तैरते हुए दिखते हैं

## 🚀 How to Use / कैसे उपयोग करें

### Option 1: Direct Use
1. Repository को clone करें:
```bash
git clone https://github.com/manishakumarim7257/Sadi-invitation-card-.git
```

2. `index.html` को अपने browser में खोलें
3. "✉ आमंत्रण पत्र खोलें" बटन क्लिक करें
4. 5 सेकंड के बाद आमंत्रण कार्ड खुल जाएगा

### Option 2: Online Use
- GitHub Pages से सीधे access करें
- किसी भी browser में खोलें

## 🎨 Customization / अनुकूलन

### तारीखें बदलने के लिए:
```html
<!-- index.html में खोजें और बदलें -->
<div class="event-date">3 जनवरी 2026, शनिवार</div>
```

### नाम बदलने के लिए:
```html
<div class="names-inside">आपका नाम</div>
<div class="couple-name">दूल्हा का नाम</div>
```

### रंग बदलने के लिए:
```css
/* CSS में ये color codes खोजें और बदलें */
#800000  /* Dark Red */
#b30000  /* Bright Red */
#ffcc00  /* Golden Yellow */
```

### संगीत बदलने के लिए:
```html
<!-- index.html में यह लाइन खोजें -->
<source src="आपके_संगीत_का_URL" type="audio/mpeg">
```

## 📝 How to Customize / अनुकूलन करने के तरीके

### Couple Names Change करने के लिए:
```html
दिपाली को अपना नाम दें
दीपक को अपना नाम दें
```

### Dates Update करने के लिए:
```html
3 जनवरी 2026 को अपनी तारीख दें
5 जनवरी 2026 को अपनी तारीख दें
```

### Location Update करने के लिए:
```html
निवास ग्राम-सेमरिया, जिला-कोरबा (छ.ग.)
को अपना स्थान दें
```

### Colors Change करने के लिए:
CSS में `linear-gradient` और color codes बदलें
